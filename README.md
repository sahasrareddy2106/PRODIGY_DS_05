## 🚦 Road Traffic Accident Data Analysis

## 📌 Task Objective
Analyze road traffic accident data to identify patterns related to time, weather, and road conditions.

## 📁 Dataset
- RTA Dataset.csv

## 🛠️ Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📊 Analysis
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization

## 📈 Key Insights
- Weather impacts accidents
- Road conditions affect severity
- Peak hours show more accidents
